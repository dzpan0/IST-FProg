# Overview

This project was developed as part of the Foundations of Programming (FProg) course at IST-UL.

## Proj1

The objective of the first part was to introduce us to the concept of programming. We has to recreate the logic of the board game Go at an high level and no rules.

## Proj2

At this part we had to develop the game in itself: be able to actually play it in the terminal.
